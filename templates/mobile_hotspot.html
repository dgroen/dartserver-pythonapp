Manual Hotspot Setup

To manually create a hotspot on your device:

Android Instructions
  1. Go to Settings ‚Üí Network & Internet ‚Üí Hotspot & tethering
  2. Tap "Wi-Fi hotspot"
  3. Set the hotspot name to your dartboard ID
  4. Set the password from your configuration
  5. Turn on the hotspot
iOS Instructions
  1. Go to Settings ‚Üí Personal Hotspot
  2. Turn on "Allow Others to Join"
  3. Note: iOS doesn't allow custom hotspot names
  4. You'll need to manually connect the dartboard to your iPhone's hotspot
